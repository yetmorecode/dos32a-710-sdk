2.10 - DPMI function 000Bh - Get Descriptor

Copies the descriptor table entry for the specified selector into an 8-byte buffer.

 

In: AX = 000Bh
BX = selector
ES:EDI = selector:offset of 8-byte buffer
Out:

if successful:
CF clear
the buffer pointed to by ES:EDI contains descriptor

if failed
CF set
AX = error code

8022h - invalid selector
