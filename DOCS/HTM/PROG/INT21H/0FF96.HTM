DOS/32 Advanced DOS Extender - Programmer's Reference

3.54 - DOS function 0FF96h - DOS/32 Advanced Free Low Memory Block

 

In: AX = 0FF96h
ESI = handle of previously allocated memory block
Out:

if successful:
CF clear

if failed:
CF set

Notes:

a) This call is specific to DOS/32 Advanced DOS Extender only, and is not supported by standard DOS.

b) This function works in exactly the same way as extended DOS function 0FF92h except that it frees a DOS memory block.
