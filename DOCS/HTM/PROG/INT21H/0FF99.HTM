DOS/32 Advanced DOS Extender - Programmer's Reference

3.57 - DOS function 0FF99h - DOS/32 Advanced Unmap Physical Memory

 

In: AX = 0FF99h
EBX = linear address returned by function 0FF98h
Out:

if successful:
CF clear

if failed:
CF set

Notes:

a) This call is specific to DOS/32 Advanced DOS Extender only, and is not supported by standard DOS.

b) This function is equivalent to DPMI function 0801h except that it uses 32-bit register instead of 16:16-bit pair of registers.
