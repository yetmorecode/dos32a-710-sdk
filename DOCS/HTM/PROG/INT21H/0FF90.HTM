DOS/32 Advanced DOS Extender - Programmer's Reference

3.48 - DOS function 0FF90h - DOS/32 Advanced Get Free High Memory Info

 

In: AX = 0FF90h
Out: EAX = size of the largest free memory block in bytes

Notes:

a) This call is specific to DOS/32 Advanced DOS Extender only, and is not supported by standard DOS.

b) This function returns information about free extended memory and is equal to DPMI function 0500h except that it returns only the first field of structure in the EAX register.
