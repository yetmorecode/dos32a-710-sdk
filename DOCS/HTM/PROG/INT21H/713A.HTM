3.35 - DOS function 713Ah - Win95 LFN Remove Directory

 

In: AX = 713Ah
DS:EDX = pointer to ASCIIZ long name of directory to remove
Out:

if successful:
CF clear

if failed:
CF set
EAX = DOS error code
