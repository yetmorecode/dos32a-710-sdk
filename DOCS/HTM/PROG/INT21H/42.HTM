3.19 - DOS function 42h - Set Current File Position

 

In:

AH = 42h
AL = origin of move:

00h = start
01h = current
02h = end

ECX:EDX = file position

Out:

if successful:
CF clear
EDX:EAX = current file position

if failed:
CF set
EAX = DOS error code
